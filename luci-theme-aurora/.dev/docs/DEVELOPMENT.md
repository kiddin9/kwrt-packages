# Quick Setup

Requires **[Node.js v20.19+](https://nodejs.org/en/download)** and **pnpm** .

### **Bash**

```bash
# Setup
git clone git@github.com:eamonxg/luci-theme-aurora.git
cd luci-theme-aurora/.dev/
# Enable pnpm
corepack enable && corepack prepare
pnpm i

# Configure HOST & Run
cp .env.example .env
# EDIT VITE_OPENWRT_HOST in the .env file to your OpenWrt LuCI web UI address
pnpm dev
```
